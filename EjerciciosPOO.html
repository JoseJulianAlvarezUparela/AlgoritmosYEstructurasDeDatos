Ejercicios de Programacion Orientada a Objetos

  • Ejercicio 1: Crea una clase `Rectángulo` con propiedades de ancho y altura, y métodos para calcular el área y el perímetro.
  • Ejercicio 2: Implementa una clase `CuentaBancaria` con propiedades de saldo y métodos para depositar y retirar dinero.
  • Ejercicio 3: Crea una clase `Vehículo` con propiedades de marca y modelo, y métodos para obtener información y conducir.
  • Ejercicio 4: Implementa una clase `Estudiante` con propiedades de nombre y calificaciones, y métodos para calcular el promedio y obtener el estado académico.
  • Ejercicio 5: Crea una clase `Calculadora` con métodos para realizar operaciones matemáticas básicas.
  • Ejercicio 6: Implementa una clase `Punto` con propiedades de coordenadas x e y, y métodos para calcular la distancia a otro punto y el cuadrante en el que se encuentra.
  • Ejercicio 7: Crea una clase `Producto` con propiedades de nombre y precio, y métodos para aplicar descuentos y calcular el precio final.
  • Ejercicio 8: Implementa una clase `Libro` con propiedades de título, autor y año de publicación, y métodos para obtener la información completa.
  • Ejercicio 9: Crea una clase `Empleado` con propiedades de nombre, salario y nivel, y métodos para calcular el salario anual y obtener el nivel de experiencia.
  • Ejercicio 10: Implementa una clase `Tienda` con propiedades de nombre y productos, y métodos para agregar productos, calcular el total de ventas y obtener el producto más vendido.
  • Ejercicio 11: Crea una clase `Triángulo` con propiedades de longitud de lados, y métodos para calcular el perímetro y el tipo de triángulo.
  • Ejercicio 12: Implementa una clase `ListaTareas` con propiedades de tareas y métodos para agregar tareas, marcar como completadas y obtener la lista de tareas pendientes.
  • Ejercicio 13: Crea una clase `Circulo` con propiedades de radio y métodos para calcular el área y la circunferencia.
  • Ejercicio 14: Implementa una clase `Agenda` con propiedades de contactos y métodos para agregar contactos, buscar por nombre y eliminar contactos.
  • Ejercicio 15: Crea una clase `Fracción` con propiedades de numerador y denominador, y métodos para realizar operaciones aritméticas básicas.
  • Ejercicio 16: Implementa una clase `EquipoFútbol` con propiedades de nombre y jugadores, y métodos para agregar jugadores, calcular la edad promedio y obtener el jugador más joven.
  • Ejercicio 17: Crea una clase `CorreoElectrónico` con propiedades de remitente, destinatario, asunto y cuerpo, y métodos para enviar y recibir correos.
  • Ejercicio 18: Implementa una clase `FiguraGeométrica` con métodos para calcular el área y el perímetro, y crea clases hijas para representar diferentes figuras geométricas.
  • Ejercicio 19: Crea una clase `CarritoCompra` con propiedades de productos y métodos para agregar productos, calcular el total y aplicar descuentos.
  • Ejercicio 20: Implementa una clase `RedSocial` con propiedades de usuarios y métodos para agregar usuarios, buscar por nombre y enviar mensajes.
  • Ejercicio 21: Crea una clase `Pila` con métodos para agregar elementos, eliminar elementos y verificar si está vacía.
  • Ejercicio 22: Implementa una clase `Banco` con propiedades de clientes y métodos para agregar clientes, realizar transferencias y calcular el saldo total.
  • Ejercicio 23: Crea una clase `Cronómetro` con métodos para iniciar, detener y reiniciar el cronómetro.
  • Ejercicio 24: Implementa una clase `Mascota` con propiedades de nombre y edad, y métodos para calcular la edad en años humanos y obtener la información completa.
  • Ejercicio 25: Crea una clase `Traductor` con propiedades de diccionario y métodos para agregar palabras, buscar traducciones y traducir frases.
